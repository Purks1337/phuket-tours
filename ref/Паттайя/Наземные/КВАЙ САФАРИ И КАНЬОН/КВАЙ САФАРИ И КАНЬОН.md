**КВАЙ САФАРИ + КАНЬОН ЛЕТАЮЩЕГО ДРАКОНА**

Стоимость: 2900/1900 бат (ростом 90-120 см)

Ребёнок ростом до 90 см бесплатно

**ОПИСАНИЕ**

**Квай Сафари + Каньон Летающего Дракона – это одна из самых насыщенных и необычных экскурсий из Паттайи за 1 день.**

**Здесь сочетается сразу несколько форматов:**

• **сафари с дикими животными**

• **уникальные буддистские места**

• **природные локации и пещеры**

• **культовый мост через реку Квай**

• **и финал – закат на стеклянном мосту над рекой**

**Это не стандартная экскурсия, а полноценное путешествие с разными эмоциями: от адреналина на сафари до спокойной атмосферы храмов и мощного финала с видом на закат.**

**И главное – без шопинга, только впечатления.**

**САМЫЕ ЯРКИЕ МОМЕНТЫ**

• Каньон Летающего Дракона с высеченной статуей Будды  
• Подъём на гольф-карах по каньону  
• Сафари с тиграми, львами и жирафами  
• Контакт с животными в зоопарке  
• Пещерный храм с проходом через гору  
• Храм с гигантской статуей Гуаньинь  
• Исторический мост через реку Квай  
• Закат на стеклянном мосту – главный вау-момент  
• Без шопинга – только насыщенная программа

**МАРШРУТ**

**05:30 – 06:00** Сбор из отелей**.** Комфортный трансфер по Паттайе.

**06:00** Выезд. Дорога по новой скоростной магистрали M-81.

**07:30** Остановка на завтрак. Завтрак берётся с собой из отеля.

**09:00** Каньон Летающего Дракона. Уникальное место с гигантской статуей Будды, высеченной в скале. Подъём на гольф-карах вдоль каньона к священному тоннелю внутри горы.

**11:30** Сафари на автобусе. Проезд через зоны обитания животных: тигры, львы, чёрные пантеры, жирафы, верблюды. Животные подходят прямо к автобусу.

**12:30** Контактный зоопарк. Возможность пообщаться с животными и сделать фото.

**14:00** Обед. Комплексный обед (русская кухня).

**15:00** Пещерный храм в горе Пхун. Прогулка через систему из 5 пещер сквозь гору с выходом к реке Квай.Потрясающие виды и атмосфера.

**16:00** Храм Метта Дхарма. Огромная 12-метровая статуя Тысячерукой Гуаньинь, вырезанная из тикового дерева.

**17:30** Мост через реку Квай. Историческое место с сильной атмосферой.

**18:00** Закат на стеклянном мосту. Финальная точка тура – смотровая площадка над рекой Квай.Один из самых красивых моментов всей экскурсии.

**18:30 – 22:00** Возвращение в Паттайю по скоростной трассе с одной остановкой в пути.

**Тур включает**: обед, трансфер из отеля и обратно, русский гид, страховка.

**Что взять с собой**: Фото/видео камера, деньги на сувениры и личные расходы, закрытая одежда для посещения храма, дождевики на случай плохой погоды (в сезон дождей).

***Примечание****:****ДНИ ПРОВЕДЕНИЯ: Понедельник / Среда / Пятница / Воскресенье***
